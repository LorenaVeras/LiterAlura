
# 📚 Literalura - Sistema de Gestão de Livros e Autores 📖

Bem-vindo ao **Literalura**, um sistema que permite pesquisar livros e autores, registrar novos livros e visualizar informações detalhadas sobre obras literárias. O projeto faz uso da API **Gutendex** para obter informações de livros e fornece funcionalidades para gerenciamento de livros e autores em um banco de dados.

## 🚀 Funcionalidades

- **Buscar por título**: Encontre livros na API Gutendex e registre-os no sistema.
- **Livros registrados**: Visualize a lista de livros já registrados no sistema.
- **Autores registrados**: Veja todos os autores que têm livros registrados.
- **Livros por idioma**: Encontre livros disponíveis em diferentes idiomas.
- **Sair**: Finaliza o programa.

## 🛠 Tecnologias

- **Java**: Linguagem de programação.
- **Spring Framework**: Framework utilizado para facilitar a criação e estruturação do projeto.
- **API Gutendex**: Fonte para pesquisa de livros.


## 🎮 Exemplo de Uso

Após executar o programa, você verá o seguinte menu de opções:

```
----------------------------
Escolha o número de sua opção:
1- Buscar por título
2- Livros registrados
3- Autores registrados
4- Por idioma
5- SAIR                                 
----------------------------
```

Escolha a opção desejada para interagir com o sistema.
